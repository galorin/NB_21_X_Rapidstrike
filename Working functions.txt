Single Fire mode
===========================
  options on default screen
    Flywheel speed - btn1
	Darts per pull - btn2 - .5sec hold - full auto mode
	Burst ROF      - btn3
	
	On option select
	 but1 - decrement - .5sec hold - select minimum
	 but2 - back/cancel
     but3 - increment - .5 sec hold select max
	
Full Auto Mode
===========================

  options on default screen
    Flywheel speed - btn1
    Burst mode     - btn2
	RoF            - btn3
	
	On option select
	 but1 - decrement - .5sec hold - select minimum
	 but2 - back/cancel 
     but3 - increment - .5 sec hold select max
